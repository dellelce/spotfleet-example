# spotfleet-example

setting up a spot fleet with terraform

## Usage

All activity is driven by a ```Makefile```:

all commands are executed through ```Gnu Make```, example:

```
make help
```

|----------------------|---------------------------------|
| target               | Description                     |
|----------------------|---------------------------------|
| help                 | Help message                    |
| terrabin             | Install Terraform locally       |
| keys                 | Generate SSH keys               |
|----------------------|---------------------------------|
